## 摘要
测试时自适应（TTA）已成为提升视觉语言模型（如CLIP）在下游任务中性能的流行范式。在现有基于CLIP的TTA方法中，测试时提示调优（TPT）是一项开创性工作，它通过多次测试时扩充来优化文本提示，并且至今仍是一个强大的基线。在本工作中，我们重新审视了TPT，并揭示了其优化可以解释为隐式地从自生成伪标签中学习。基于这一视角，我们提出了一种统一的自集成框架（USE），以确保优化和推理阶段之间的一致性。在优化过程中，我们引入了一种简单而有效的自集成（SE）策略，该策略自适应地强调测试图像本身而非其扩充视图，以获得更可靠的伪标签。为了充分利用扩充的潜力，我们还在推理时应用相同的策略，统一了两个阶段的目标。值得注意的是，SE还可以作为一种轻量级的无优化TTA方法。跨多个数据集的广泛实验表明，SE和USE分别优于它们的对应方法。此外，当与现有TTA方法集成时，SE产生了一致的性能提升。代码可在https://github.com/sirujiang/USE获取。

## Abstract
Test-time adaptation (TTA) has emerged as a popular paradigm for improving the performance of vision-language models (e.g., CLIP) on downstream tasks. Among existing CLIP-based TTA methods, Test-Time Prompt Tuning (TPT) is a pioneering work that optimizes textual prompts using multiple test-time augmentations and remains a strong baseline to date. In this work, we revisit TPT and reveal that its optimization can be interpreted as implicitly learning from self-generated pseudo labels. Building on this perspective, we propose a unified self-ensembling framework (USE) that ensures consistency between the optimization and inference stages. During optimization, we introduce a simple yet effective self-ensembling (SE) strategy that emphasizes the test image itself over its augmented views adaptively to obtain more reliable pseudo labels. To fully exploit the potential of augmentations, we further apply the same strategy at inference time, unifying the objectives of both stages. Notably, SE can also act as a lightweight optimization-free TTA method. Extensive experiments across multiple datasets demonstrate that SE and USE outperform their counterparts, respectively. Furthermore, SE yields consistent performance gains when integrated with existing TTA methods. The code is available at https://github.com/sirujiang/USE.

- **问题**：视觉语言模型（如 CLIP）在下游任务中面对分布偏移时性能下降。测试时自适应（TTA）通过无标签测试样本优化模型，但现有代表性方法 TPT（Test-Time Prompt Tuning）在优化阶段通过多个增强视图的预测平均来生成伪标签并最小化边际熵，而在推理阶段仅使用弱增强（原始图像）进行预测，导致两阶段目标不一致，限制了伪标签质量和最终性能。
- **整体含义**：本文从自学习视角重新解释 TPT，发现其优化可等价于隐式地让增强样本向自生成伪标签对齐。基于此，提出统一自集成框架 USE，使优化和推理阶段采用相同的自适应集成策略，提升伪标签可靠性，并最终提高分类准确性。

### 2. 方法论：核心思想、关键技术细节
- **核心思想**：将 TPT 的边际熵解耦为反向交叉熵 + KL 散度，固定伪标签后优化等价于最小化反向交叉熵，即驱动增强视图向自生成伪标签学习。因此，关键是如何获得更好的伪标签。
- **关键技术细节**：
    1. **自集成伪标签估计（SE）**：在优化阶段，保留弱增强（原始图像）预测 `p_w`，并计算选定的强增强平均预测 `p_s`。定义相对熵排名 `δ`（弱增强在强增强中的熵排名比例），通过线性映射得到权重 `β = 0.5 + γ(δ-0.5)`（γ=0.4，β∈[0.3,0.7]），最终伪标签 `q = β·p_w + (1-β)·p_s`。该策略自适应强调可靠的原图，避免其被等权平均稀释。
    2. **优化目标**：使用反向交叉熵 `L_rce = RCE(¯p; q)` 对所有选定视图（S 集合）进行优化，其中 `¯p` 是 S 中所有视图的平均预测。
    3. **推理阶段**：同样使用 SE 策略（与优化阶段相同的 β 和视图）计算最终预测 `ŷ = β·p_w + (1-β)·p_s`，确保目标一致。
    4. **加速技巧**：当弱增强预测与所有选定强增强预测一致时，跳过优化，降低计算开销。
- **算法流程**：输入测试图像 → 生成 63 个增强（弱 + 强） → 计算各视图预测和熵 → 根据熵阈值选择高置信度视图 → 计算 δ 和 β → 构造伪标签 q → 用反向交叉熵更新文本提示 → 用更新后的模型重新计算预测，再次应用 SE 得到最终输出。

### 6. 主要结论与发现
- **SE 作为无优化方法**：在多个数据集上达到最佳或次佳性能，例如在 ViT-B/16 细粒度平均 65.02%，优于 MTA (64.77%) 和 ZERO (63.86%)。

### 7. 优点
- **理论创新**：从伪标签学习角度重新解释 TPT，揭示其优化本质，为改进提供清晰指导。
- **方法简洁有效**：SE 仅需简单的熵排名加权，无需额外网络或训练，易于实现和集成。
- **统一框架**：首次在 TTA 中统一优化和推理目标，避免阶段不一致带来的性能损失。
- **轻量且可扩展**：SE 可作为零开销增强模块，与其他 TTA 方法无缝结合，带来一致增益。
- **鲁棒性**：对超参数 γ 和不同缩放函数不敏感，在多种架构和数据集上表现稳定。
