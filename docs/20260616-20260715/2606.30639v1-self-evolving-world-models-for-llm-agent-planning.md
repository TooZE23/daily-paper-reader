## 摘要
世界模型为长视界LLM智能体提供了一种原则性的方法来赋予其预见能力：在执行前预测行动后果。然而，不可靠的预见可能被忽略、误用，甚至降低下游决策质量。本文介绍了WorldEvolver，一种自我进化的世界模型框架，它在保持下游智能体和所有模型参数冻结的情况下，修正其部署时的上下文。WorldEvolver集成了三个模块：（i）情景记忆，通过基于检索的模拟利用真实行动转移；（ii）语义记忆，从预测-观察不匹配中提取持久启发式规则；（iii）选择性预见，在将低置信度预测整合到智能体推理上下文之前进行过滤。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上测量世界模型预测准确性，在AgentBoard上测量下游智能体成功率。大量实验表明，WorldEvolver在三个骨干网络上实现了最高的预测准确性，并在下游智能体成功率上领先于其他世界模型基线，证明了测试时记忆修正同时增强了预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

#### 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：长视界 LLM 智能体依赖世界模型提供“预见”（行动后果预测）来辅助规划，但部署环境中世界模型的预测往往不可靠，导致智能体忽略、误用甚至被错误预测误导，反而降低决策质量。
- **研究动机**：现有方法要么冻结世界模型（无法适应分布偏移），要么通过梯度更新（成本高、易遗忘、不适用于在线部署）。此外，不完全可信的预见可能比没有预见更差（如图 2 的 oracle 实验所示）。
- **整体含义**：本文提出一种**自我进化**的世界模型框架 WorldEvolver，在不更新模型参数、不改变下游智能体的情况下，通过**动态修正部署时的上下文记忆**来提升预测的保真度和对智能体规划的正向影响。

#### 2. 方法论：核心思想、关键技术细节
- **核心思想**：将世界模型的适应性转移到**外部记忆**上，而非模型权重。通过三类记忆模块在推理时自动积累和利用经验，从而持续改进预测。
- **关键技术细节**：
  - **Episodic Memory（情景记忆，M_E）**：存储已执行的真实转移（observation, action, observation’），在预测当前动作结果时，通过 Jaccard 相似度检索最相似的 k 条历史转移作为上下文示例（exploitation）。
  - **Semantic Memory（语义记忆，M_S）**：当预测与真实观察不符时，利用 LLM 批评器将不匹配转化为可复用的启发式规则（如“检查操作不会移动物体”），并赋予证据分数，只有正分数规则保留，不断更新（exploration）。
  - **Selective Foresight（选择性预见）**：计算预测输出的几何平均 token 概率作为置信度，只有当置信度高于阈值 τ 时，才将预测暴露给下游智能体；否则不提供任何预见（即抽象）。
- **算法流程（Algorithm 1）**：
  1. 智能体根据当前状态采样一个草稿动作。
  2. 从 Episodic Memory 中检索与动作相似的转移。
  3. 世界模型结合检索到的记忆和语义规则生成预测及置信度。
  4. Selective Foresight 决定是否传递预测。
  5. 智能体基于（状态 + 可选预见）采样最终执行动作。
  6. 如果最终动作不同于草稿动作，重新用最终动作查询世界模型。
  7. 执行动作后，将真实转移加入 Episodic Memory。
  8. 若预测与观察不符（经因子化三元组比较），用 LLM 批评器生成规则更新 Semantic Memory。

#### 7. 优点
- **训练无关**：无需任何梯度更新或离线微调，完全基于推理时上下文拓展。
- **模块化与可解释性**：Episodic Memory 提供具体案例，Semantic Memory 提供可读规则，Selective Foresight 提供可调阈值，原理清晰，便于调试。
- **鲁棒性**：通过置信度过滤避免坏预测误导智能体，实验证明即使在强骨干下也不会降低性能。
- **计算效率**：相较于 ITP-I 的多步想象，WorldEvolver 仅需单步额外推理，运行时开销可控（约 40%–50% 增加，但准确率提升显著）。
- **泛化性**：在两种不同环境（家庭和科学实验室）、两种不同智能体策略、四个骨干上均有效。

#### 8. 不足与局限
- **实验覆盖有限**：仅评估了 ALFWorld 和 ScienceWorld 两个文本环境，未涉及网页导航、机器人、多模态等更复杂领域。
- **置信度估计依赖 token 概率**：对闭源 API 不适用（如 GPT 系列），并且作者承认置信度与准确性的相关性可能随环境变化。
- **Episodic Memory 检索质量**：仅基于动作的 Jaccard 相似度，未利用状态信息，可能丢失关键上下文。
- **Semantic Memory 规则质量**：依赖 LLM 批评器生成规则，可能引入噪声或错误规则（尽管有证据分数过滤）。
- **参数固定**：阈值 τ 由校准曲线确定且固定，未探索动态自适应机制。
- **缺乏开放世界或动态持续学习**：任务间记忆共享是手动设定的，未在更长的多任务序列中验证。
