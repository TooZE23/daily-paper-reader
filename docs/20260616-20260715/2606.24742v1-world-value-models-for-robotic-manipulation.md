## 摘要
通用价值模型在从大规模、混合质量数据中扩展机器人策略学习中发挥着关键作用。从数学上讲，准确的价值估计需要深度的时间理解，要求模型既利用历史上下文来确立当前信念，又能对未来结果进行规划。然而，大多数现有的机器人价值模型都基于视觉-语言模型（VLM）骨干网络，这些网络主要针对静态或时间稀疏的视觉观察进行预训练，缺乏价值估计所需的时间建模能力。与VLM不同，世界模型天然擅长时间建模和未来规划，使其成为学习可泛化价值函数的理想基础。基于这一洞察，我们将世界模型与价值估计相结合，构建了一种新的通用机器人价值模型——世界价值模型（WVM），该模型能够提供准确的任务进展以评估数据质量。在标准基准上，WVM取得了最先进的（SOTA）价值序相关性（VOC）结果。为了补充仅包含专家数据的标准评估套件，我们进一步引入了Suboptimal-Value-Bench，这是一个多体态基准，包含800条带有高保真度、人工标记帧注释的次优轨迹。我们的评估表明，WVM在Suboptimal-Value-Bench上保持了其最先进性能，证明了其在处理专家数据和次优数据方面的鲁棒性。当用于策略学习时，WVM在模拟和实际部署中提升了各种策略提取方法的操作性能，为从混合质量数据中学习提供了稳健的指导。

## Abstract
Generalist value models play a pivotal role in scaling robotic policy learning from large-scale, mixed-quality data. Mathematically, accurate value estimation demands deep temporal understanding, requiring models to both ground the current belief using historical context and plan over future outcomes. However, most existing robotic value models are built on Vision-Language Model (VLM) backbones that are pretrained primarily on static or temporally sparse visual observations, lacking the requisite temporal modeling capabilities for value estimation. Unlike VLMs, world models naturally excel at temporal modeling and future planning, making them ideal foundations for learning generalizable value functions. Driven by this insight, we marry world models with value estimation to construct a new generalist robotic value model, World Value Model (WVM), that offers accurate task progressions to assess data quality. On standard benchmarks, WVM delivers state-of-the-art (SOTA) Value-Order Correlation (VOC) results. Complementing standard evaluation suites that contains only expert data, we further introduce Suboptimal-Value-Bench, a multi-embodiment benchmark consisting of 800 suboptimal trajectories with high-fidelity, human-labeled frame annotations. Our evaluations show that WVM maintains its SOTA performance on Suboptimal-Value-Bench, establishing its robustness in handling both expert and suboptimal data. When deployed for policy learning, WVM improves manipulation performance across various policy extraction approaches in both simulated and real-world deployment, providing robust guidance for learning from mixed-quality data.

## 1. 论文的核心问题与整体含义

- **研究动机**：机器人泛化价值模型（generalist value models）对于从大规模、混合质量数据中学习策略至关重要。但现有价值模型普遍基于视觉-语言模型（VLM）骨干，这些VLM在静态或时间稀疏的视觉数据上预训练，缺乏密集时间建模能力。而准确的价值估计需要同时理解历史上下文和规划未来结果（时间双向建模）。
- **背景意义**：世界模型（world models）天然擅长时间动力学建模和未来预测，本文核心洞察是将其时空先验重用于价值学习，解决VLM的瓶颈。作者构建了世界价值模型（World Value Model，WVM），弥补现有价值模型在时间理解上的缺陷，并从混合质量数据中提供更准确的任务进度评估。

## 2. 方法论

### 核心思想
- 将价值估计重新表述为**分块预测问题**：给定长度为h的观测序列和语言指令，预测对应h帧的归一化任务进度，而不是单个标量值。这样能捕捉局部的进展曲线（如平台、下降），反映时间动态。
- 利用预训练的视频世界模型作为特征提取器，并通过**Mixture-of-Transformers (MoT)** 架构让价值流选择性关注视频潜在特征，同时最小化对视频生成流的干扰。

### 关键技术细节
- **视频流**：基于Wan2.2的视频VAE和视频DiT，输入长度为2h+1的视频片段（包含1帧前缀、h帧当前观测、h帧目标未来帧）。VAE编码后，丢弃前缀潜在、保留当前上下文、损坏未来潜在用于视频生成去噪。
- **价值流**：轻量级DiT，从噪声价值令牌预测价值块，通过MoT多模态自注意力关注视频DiT中间特征。注意力掩码是不对称的：价值令牌可注意视频令牌，但视频令牌不关注价值令牌。
- **训练目标**：使用流匹配（flow matching）损失，分别应用于未来视频潜在和价值块，总损失 `L = L_value + λ L_video`。价值块被建模为分布性预测，通过流匹配生成连续密度，避免标量或分类头的信息丢失。
- **训练技巧**：
  - **前缀随机化**：以概率p将前缀价值替换为[0,1]均匀随机标量，防止价值流依赖前缀作为捷径，类似分类器无引导的dropout。
  - **视频倒带（ReWiND）**：对每个窗口随机应用上升、平台、下降三种时间模式，重标对应价值，让模型暴露于次优局部进展。
- **推理**：使用显式欧拉求解器单步去噪价值块，并采用重叠窗口平均方案以保持连续性。

### 公式/算法流程（文字说明）
1. 给定h帧观测和语言指令，构造视频片段（前缀+当前+未来）。
2. 视频VAE编码为潜在，当前潜在保留，未来潜在加噪。
3. 视频DiT对加噪未来潜在去噪；价值DiT从噪声价值令牌去噪，过程中通过MoT关注视频DiT的中间特征。
4. 流匹配损失同时优化视频恢复和价值预测。
5. 训练时应用前缀随机化和视频倒带增强。
6. 推理时单步去噪得到价值块，重叠平均得到逐帧价值。

## 6. 主要结论与发现

1. **WVM在价值估计上达到SOTA**：
   - Suboptimal-Value-Bench上Hesitation-RMSE平均0.05（优于最接近基线GVL/Robometer的0.14），Retry-VOC平均0.78（优于GVL的0.62）。

2. **视频共训练至关重要**：去掉视频损失后Hesitation-RMSE上升、Retry-VOC下降；冻结视频权重导致严重退化。
3. **前缀随机化平衡了捷径抑制和跨片段连续性**：p=0.5最优，p=0过度依赖前缀（Expert-VOC虽高但次优检测差），p=1则破坏连续性。
4. **流匹配价值头优于分类头**：HL-Gaussian头在所有指标上劣于流匹配，说明连续密度分布对保持局部价值差异更重要。
5. **下游策略显著提升**：使用WVM指导AWR或Filtered BC，在仿真和真实世界中性能均超过普通BC，验证了WVM能有效区分真实任务进展与次优行为。

## 7. 优点

- **方法创新**：首次将世界模型的时空先验重用于通用价值学习，利用其固有的时间建模能力克服VLM的局限。
- **架构精巧**：通过MoT实现价值流与视频流的耦合，保持视频生成能力同时提取价值特征；不对称注意力屏蔽确保视频流不被价值流干扰。
- **训练设计**：前缀随机化和视频倒带的组合有效防止了价值流取巧，并提供了次优行为监督。
- **评估全面**：不仅用标准Expert-VOC，还构建了包含人类标注的次优轨迹基准Suboptimal-Value-Bench，弥补了之前只评估专家轨迹的不足。该基准包含3种实体15个任务，覆盖犹豫和重试两种典型次优模式。
- **下游验证**：将价值评估直接用于策略学习，证明了真实效用，而非仅停留在指标上。

## 8. 不足与局限

- **数据规模有限**：作者承认由于计算限制，训练数据集规模不够大，导致WVM在面对全新任务和场景时zero-shot能力受限。
- **基准覆盖范围窄**：Suboptimal-Value-Bench主要集中在拾取-放置任务，缺乏灵巧操作和长时延操作（如穿针引线、多阶段装配），这些任务的价值评估可能更复杂。未来需扩展到更广泛的任务（如灵巧手操控、长时间任务）。
- **算力成本高**：5.7B参数模型在32张A100上训练40小时，且价值部分需0.7B额外参数，推理时虽单步去噪但仍有较大计算开销，可能影响实时性。
- **对比基线选择**：虽然对比了多种基线，但其中一些（如TopReward、RoboReward）是基于零样本或小样本设置，可能对VLM基线不公平（WVM使用了大规模视频预训练）。作者也提到Expert-VOC在EgoDex上RoboReward略优于WVM，暴露了该指标的局限性。
- **下游实验规模小**：每个任务仅10-50条次优轨迹，策略提升的泛化性有待进一步验证（如在更多任务、更多机器人实体上测试）。
- **价值定义假设**：将价值与归一化任务进度等同，假设稀疏奖励设置（即任务完成前均为-1），可能不适用于更复杂的奖励结构（如连续奖励、多目标奖励）。
