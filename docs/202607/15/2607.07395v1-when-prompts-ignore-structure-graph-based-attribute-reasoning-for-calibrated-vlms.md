## 摘要
可靠的置信度估计仍然是视觉语言模型（VLM）中测试时自适应的一个关键限制，其中提示调整提高了零样本准确率，但往往由于熵驱动的过度自信而降低了校准度。先前的方法通过使用大语言模型衍生的类属性和对比正则化来缓解这一问题，但这些方法独立地处理属性，忽略了它们的关系结构。我们提出了ARGTCA，它将（类，属性）对表示为符号属性图中的节点，并使用对比目标训练图注意力网络（GAT），以生成捕捉属性间依赖关系的结构信息嵌入。我们引入了两种属性选择策略：ARGTCA-DIV用于类内多样性，ARGTCA-DISC用于类间区分性。在九个基准上的实验表明，ARGTCA-DIV将平均期望校准误差（ECE）相比基线降低了约37%，而ARGTCA-DISC始终作为第二优的变体，将平均ECE相比基线降低了约17%。这些结果表明，对符号属性交互进行建模为VLM中可靠的测试时自适应提供了一种有原则的方法。

## Abstract
Reliable confidence estimation remains a key limitation of test-time adaptation in vision-language models (VLMs), where prompt tuning improves zero-shot accuracy but often degrades calibration due to entropy-driven overconfidence. Prior approaches mitigate this using LLM-derived class attributes and contrastive regularization, yet treat attributes independently, ignoring their relational structure. We propose ARGTCA, which represents (class, attribute) pairs as nodes in a Symbolic Attribute Graph and trains a Graph Attention Network (GAT) using contrastive objectives to produce structurally informed embeddings that capture inter-attribute dependencies. We introduce two attribute selection strategies: ARGTCA-DIV for intra-class diversity and ARGTCA-DISC for inter-class discrimination. Experiments across nine benchmarks show that ARGTCA-DIV reduces average Expected Calibration Error (ECE) by approximately ~37% over baselines, while ARGTCA-DISC consistently performs as the second-best variant, reducing average ECE by approximately ~17% over baselines. These results suggest that modeling symbolic attribute interactions provides a principled approach for reliable test-time adaptation in VLMs.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：视觉语言模型（VLM，如CLIP）在测试时通过提示调优（Test-time Prompt Tuning, TPT）提升零样本分类准确率，但熵最小化目标会导致模型过度自信，使预测概率严重偏离真实置信度，即校准退化（calibration degradation），表现为期望校准误差（ECE）显著升高。
- **研究背景**：
  - 校准与准确率部分解耦：不同提示在准确率相近时，ECE差异很大，说明文本特征空间的几何结构主导校准质量。
  - 现有方法（如C-TPT、O-TPT、A-TPT）通过添加特征分散损失改善几何属性，但独立处理LLM生成的属性，忽略了属性间的语义关系。
  - 直接前身TCA利用LLM生成的视觉属性初始化提示，并加入对比正则化，但其属性选择仅基于与类名的余弦相似度排序，导致两个结构失效：①同义词属性在嵌入空间高度聚集（如“rough”“bumpy”“grainy”）；②跨类共享属性（如“mammal”同时出现在豹、海豚、海狸中）缺乏区分力。
- **整体含义**：忽视属性间关系结构是导致校准不佳的关键因素，需要通过显式的图结构建模属性交互来提升校准可靠性。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

**核心思想**：将（类, 属性）对作为节点构建符号属性图（Symbolic Attribute Graph, SAG），通过图注意力网络（GAT）学习结构感知的属性嵌入，并据此设计两种属性选择策略，以替代TCA中基于余弦相似度的平坦选择。

**关键技术细节**：

- **符号属性图（SAG）构造**：
  - 每个类\(c_k\)有\(M\)个LLM生成的属性\(a_j\)，共\(N=K\times M\)个节点。
  - 边分为两类：
    - 类内边\(E_{\text{intra}}\)：连接同一类的所有属性节点（\(c_i = c_j\)），促进类内互补性。
    - 类间边\(E_{\text{inter}}\)：连接共享相同属性词的不同类节点（\(a_i = a_j\)），传播跨类上下文。
  - 节点初始特征：将短语“a [attribute] of a [class]”输入冻结的CLIP文本编码器，取EOS隐藏状态并\(\ell_2\)归一化得到\(\mathbf{h}_i^{(0)}\)。

- **图注意力网络（GAT）训练**：
  - 两层GAT（4头注意力，512维节点特征），ELU激活，残差连接，\(\ell_2\)归一化。
  - 训练目标：监督对比损失（Supervised Contrastive Loss）：
    \[
    \mathcal{L}_{\text{SupCon}} = \frac{1}{|A|}\sum_{i\in A}\frac{-1}{|P(i)|}\sum_{p\in P(i)}\log\frac{\exp(\tilde{\mathbf{h}}_i^\top\tilde{\mathbf{h}}_p/\tau)}{\sum_{j\neq i}\exp(\tilde{\mathbf{h}}_i^\top\tilde{\mathbf{h}}_j/\tau)}
    \]
    其中正例为同一类的节点，负例为所有跨类节点。
  - 训练后得到结构感知嵌入\(\mathbf{h}^{(L)}\)。

- **属性选择策略**（训练后离线执行，无测试时额外成本）：
  - **ARGTCA-DIV（类内多样性）**：对每个类，选择在\(\mathbf{h}^{(L)}\)空间中余弦相似度最小的属性对，即语义最互补的一对。
  - **ARGTCA-DISC（类间区分性）**：对每个属性，计算与其他所有类节点的平均余弦距离（区分度得分），为每个类选择得分最高的\(M'\)个属性。
  - 默认\(M'=2\)。

- **测试时调优（Phase 2）**：
  - 使用选中属性构造提示：\(t_{k,m} = [\text{SOS}] \oplus p \oplus a_m \oplus c_k\)，其中\(p\)为可学习的软上下文（初始化为“a photo of a”）。
  - 文本特征\(f_{k,m}=g(t_{k,m})\)。
  - 类概率通过bag-of-attributes softmax聚合（公式2）。
  - 目标函数（与TCA相同）：
    \[
    \mathcal{L}_{\text{TCA}} = \mathcal{L}_{\text{TPT}} - \alpha \mathcal{L}_{\text{inter}} + \beta \mathcal{L}_{\text{intra}}
    \]
    其中\(\mathcal{L}_{\text{inter}}\)为类质心与全局质心的平均距离（最大化），\(\mathcal{L}_{\text{intra}}\)为类内属性嵌入与类质心的平均距离（最小化）。
  - 每个测试图像：64个增强视图，选择熵最低的10%视图，执行一步AdamW更新软提示\(p\)，然后对原始图像做预测。
